  # PHAIM

  **Platform for Human-AI Integrated Management**

  An AI-native project intelligence platform that enables non-technical users
  to build and manage complex projects through structured human-AI collaboration.

  ## What is PHAIM?

  PHAIM bridges the gap between human vision and technical execution.
  It allows anyone — regardless of programming or management experience —
  to define, track, and deliver projects by working alongside AI agents.

  ## Status

  🚧 In active development. Not yet available for public use.

  ## Core concepts

  - Ideas, tasks, bugs, and decisions tracked as a knowledge graph
  - Lifecycle management: idea → task → implementation
  - Hybrid search: keyword + semantic + graph traversal
  - Human and AI agents collaborate through structured workflows
